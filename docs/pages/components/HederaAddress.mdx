# HederaAddress Component

The `HederaAddress` component displays a Hedera address with optional **Hedera account ID** resolution. When given an EVM address, it resolves and shows the native account ID (e.g. `0.0.12345`) and can show the EVM address as secondary. It includes a copy button and a link to the block explorer (HashScan for Hedera chains).

## Behavior

- **With `address` (EVM):** Resolves to Hedera account ID via the host’s `/api/hedera?evm=...` (or custom resolver). Shows account ID as primary, optional "EVM: 0x..." as secondary. While resolving, shows "Resolving Hedera Account ID…".
- **With `hederaAccountId`:** Displays the account ID as primary; if `address` is also provided and `showEvmAddress` is true, shows the EVM address as secondary.
- **Explorer link:** Uses Hedera-aware URLs (e.g. HashScan `/account/` for Hedera chains) via `getBlockExplorerAddressLink` from `@scaffold-ui/hooks`. Pass **`blockExplorerAddressLink`** to override the URL used for the primary line’s link.
- **Invalid `hederaAccountId`:** If `hederaAccountId` is non-empty but not a valid `0.0.n` pattern, the component shows an error row (invalid account ID) instead of the normal display.
- **Invalid `address`:** If `address` is present but not a valid EVM address, and there is no valid `hederaAccountId` to fall back on, the component shows an error row (invalid address).

The host app must provide `GET /api/hedera?evm=...&network=...` (or set `setHederaAccountIdResolver` from `@scaffold-ui/hooks`) for EVM → account ID resolution.

## Import

```tsx
import { HederaAddress } from "@scaffold-ui/components";
```

## Props

| Prop                       | Type                                                       | Default   | Description                                                                                                                 |
| -------------------------- | ---------------------------------------------------------- | --------- | --------------------------------------------------------------------------------------------------------------------------- |
| `address`                  | `Address`                                                  | -         | Optional EVM address. When valid, the component resolves to account ID unless `hederaAccountId` supplies a valid native ID. |
| `hederaAccountId`          | `string`                                                   | -         | Optional pre-resolved Hedera account ID (e.g. `0.0.12345`).                                                                 |
| `chain`                    | `Chain`                                                    | required  | Blockchain network (e.g. `hederaTestnet`) for resolution and explorer link.                                                 |
| `format`                   | `"short" \| "long"`                                        | `"short"` | Display format for the address (truncated vs full).                                                                         |
| `size`                     | `"xs" \| "sm" \| "base" \| "lg" \| "xl" \| "2xl" \| "3xl"` | `"base"`  | Typography and avatar size.                                                                                                 |
| `disableAddressLink`       | `boolean`                                                  | `false`   | Disable the link to the block explorer.                                                                                     |
| `avatarComponent`          | `ComponentType<{ address, size, ensImage }>`               | -         | Optional component to render an avatar (e.g. blockie) instead of the default placeholder.                                   |
| `showEvmAddress`           | `boolean`                                                  | `true`    | When account ID is primary, show "EVM: 0x..." as secondary.                                                                 |
| `style`                    | `CSSProperties`                                            | -         | Custom styles on the root wrapper (memoize to prevent re-renders).                                                          |
| `blockExplorerAddressLink` | `string`                                                   | -         | Overrides the computed block explorer URL for the primary display link.                                                     |
