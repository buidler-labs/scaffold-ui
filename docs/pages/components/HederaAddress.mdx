# HederaAddress Component

The `HederaAddress` component displays a Hedera address with optional **Hedera account ID** resolution. When given an EVM address, it resolves and shows the native account ID (e.g. `0.0.12345`) and can show the EVM address as secondary. It includes a copy button and a link to the block explorer (HashScan for Hedera chains).

## Behavior

- **With `address` (EVM):** Resolves to Hedera account ID via the host’s `/api/hedera/account` (or custom resolver). Shows account ID as primary, optional "EVM: 0x..." as secondary. While resolving, shows "Resolving Hedera Account ID…".
- **With `hederaAccountId`:** Displays the account ID as primary; if `address` is also provided and `showEvmAddress` is true, shows the EVM address as secondary.
- **Explorer link:** Uses Hedera-aware URLs (e.g. HashScan `/account/` for Hedera chains) via `getBlockExplorerAddressLink` from `@scaffold-ui/hooks`.

The host app must provide `GET /api/hedera/account?evm=...&network=...` (or set `setHederaAccountIdResolver` from `@scaffold-ui/hooks`) for EVM → account ID resolution.

## Import

```tsx
import { HederaAddress } from "@scaffold-ui/components";
```

## Props

| Prop                 | Type                                                                 | Default  | Description                                                                 |
| -------------------- | -------------------------------------------------------------------- | -------- | --------------------------------------------------------------------------- |
| `address`            | `Address`                                                            | -        | The EVM address to display. When provided, the component resolves to account ID. |
| `hederaAccountId`    | `string`                                                             | -        | Optional pre-resolved Hedera account ID (e.g. `0.0.12345`).                  |
| `chain`              | `Chain`                                                              | required | Blockchain network (e.g. `hederaTestnet`) for resolution and explorer link.  |
| `format`             | `"short" \| "long"`                                                  | `"short"`| Display format for the address (truncated vs full).                          |
| `disableAddressLink` | `boolean`                                                            | `false`  | Disable the link to the block explorer.                                      |
| `avatarComponent`    | `ComponentType<{ address, size, ensImage }>`                         | -        | Optional component to render an avatar (e.g. blockie) instead of the default placeholder. |
| `showEvmAddress`     | `boolean`                                                            | `true`   | When account ID is primary, show "EVM: 0x..." as secondary.                   |
