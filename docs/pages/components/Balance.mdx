# Balance Component

The `Balance` component displays the native token balance (e.g., ETH, MATIC, HBAR) for an address on a given chain, with support for USD conversion and interactive toggling. On **Hedera** chains (mainnet and testnet), the USD price is sourced from HBAR (CoinGecko); no extra configuration is required.

## Import

```tsx
import { Balance } from "@scaffold-ui/components";
```

## Props

| Prop             | Type            | Default                      | Description                                            |
| ---------------- | --------------- | ---------------------------- | ------------------------------------------------------ |
| `address`        | `Address`       | -                            | **Required.** The Ethereum address to show balance for |
| `chain`          | `Chain`         | First chain from [WagmiProvider](https://wagmi.sh/react/api/WagmiProvider) config or `mainnet` | Blockchain network to query                            |
| `defaultUsdMode` | `boolean`       | `false`                      | Start in USD display mode                              |
| `style`          | `CSSProperties` | -                            | Custom CSS styles (memoize to prevent re-renders)      |

### Hedera (HBAR)

On Hedera chains, pass a Hedera chain (e.g. `hederaTestnet`) and the component uses the HBAR price for USD conversion automatically.

```tsx
import { Balance } from "@scaffold-ui/components";
import { hederaTestnet } from "viem/chains";

<Balance address="0x..." chain={hederaTestnet} />
```
