# Hooks

Our components are built on top of these React hooks, which you can use directly to create your own custom Hedera components, or even combine them to build more complex ones.

## Hedera Native

- **[useMirrorNodeAccount](/hooks/useMirrorNodeAccount)** — Single source of truth for account data (identity, key type, balance) from the Mirror Node.
- **[useNativeTransaction](/hooks/useNativeTransaction)** — Send native Hedera transactions (HTS, HCS, CryptoTransfer) via the Hedera namespace; use `setNativeTransactionSigner` to inject a signer, or handle **CapabilityError** when the session can’t sign.
- **[useCreateTopic](/hooks/useCreateTopic)** — Create an HCS topic via `TopicCreateTransaction` and `useNativeTransaction`; optional `ensureReady` for wallet gating.
- **[useCreateToken](/hooks/useCreateToken)** — Create a fungible HTS token via `TokenCreateTransaction` and `useNativeTransaction`; supply `getTreasuryAccountId` for the treasury account.
- **[useHbarInput](/hooks/useHbarInput)** — Convert between HBAR (or native token) and USD; Hedera-first default.
- **[useHederaAddressInput](/hooks/useHederaAddressInput)** — Validate and resolve Hedera input as native `0.0.n` or EVM `0x…`; powers `HederaAddressInput`.
- **[useHederaAccountId](/hooks/useHederaAccountId)** — Resolve a Hedera account ID (`0.0.n`) from an EVM address via mirror node.
- **[useHederaEvmAddress](/hooks/useHederaEvmAddress)** — Resolve an EVM address from a Hedera account ID (`0.0.n`) via mirror node.

## Hedera EVM Compatible

- **[useAddress](/hooks/useAddress)** — Address display helpers and block explorer links for the given chain.
- **[useBalance](/hooks/useBalance)** — Fetch and watch balances with USD conversion toggle.

## Installation

### Peer Dependencies

Install the required peer dependencies using your preferred package manager:

:::code-group

```bash [npm]
npm install react react-dom @types/react viem wagmi @tanstack/react-query
```

```bash [pnpm]
pnpm add react react-dom @types/react viem wagmi @tanstack/react-query
```

```bash [yarn]
yarn add react react-dom @types/react viem wagmi @tanstack/react-query
```

:::

For **useCreateTopic** and **useCreateToken**, also install **`@hiero-ledger/sdk`** (peer of `@scaffold--hbar-ui/hooks`).

### Package Installation

Install Scaffold Hbar UI hooks using your preferred package manager:

:::code-group

```bash [npm]
npm install @scaffold-hbar-ui/hooks
```

```bash [pnpm]
pnpm add @scaffold-hbar-ui/hooks
```

```bash [yarn]
yarn add @scaffold-hbar-ui/hooks
```

:::
