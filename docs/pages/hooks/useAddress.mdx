# useAddress Hook

The `useAddress` hook formats EVM addresses (checksum, short form), builds a **block explorer link** for the given `chain` (HashScan-style paths on Hedera), and generates a **blockie** image URL. Pass a `chain` from `viem/chains` (e.g. `hedera`, `hederaTestnet`) for correct explorer URLs.

## Import

```tsx
import { useAddress } from "@scaffold-ui/hooks";
```

## Usage

```tsx
import { hederaTestnet } from "viem/chains";

const { checkSumAddress, blockExplorerAddressLink, isValidAddress, shortAddress, blockieUrl } = useAddress({
  address: "0xd8da6bf26964af9d7eed9e03e53415d37aa96045",
  chain: hederaTestnet,
});
```

## Parameters

| Parameter | Type                   | Default  | Description                                                                                         |
| --------- | ---------------------- | -------- | --------------------------------------------------------------------------------------------------- |
| `address` | `Address` _(optional)_ | -        | The EVM address to process.                                                                         |
| `chain`   | `Chain` _(optional)_   | See note | Used for explorer links when `address` is omitted from the hook options; prefer passing explicitly. |

## Return Values

| Property                   | Type                   | Description                                      |
| -------------------------- | ---------------------- | ------------------------------------------------ |
| `checkSumAddress`          | `Address \| undefined` | Checksummed address when input is valid.         |
| `ens`                      | `undefined`            | Reserved; not populated in this build.           |
| `ensAvatar`                | `undefined`            | Reserved; not populated in this build.           |
| `isEnsNameLoading`         | `false`                | Reserved.                                        |
| `blockExplorerAddressLink` | `string`               | Explorer URL for the address on the given chain. |
| `isValidAddress`           | `boolean`              | Whether the input is a valid EVM address.        |
| `shortAddress`             | `string \| undefined`  | Short form (`0x1234…5678`).                      |
| `blockieUrl`               | `string \| undefined`  | Deterministic blockie data URL.                  |

## Live Example

export function UseAddressBasicExample() {
  const address = "0xd8da6bf26964af9d7eed9e03e53415d37aa96045";
  const { checkSumAddress, shortAddress, blockExplorerAddressLink } = useAddress({ address });
  return (
    <div>
      <p>Short: {shortAddress}</p>
      <p>
        Full with explorer link:{" "}
        <a
          href={blockExplorerAddressLink}
          target="_blank"
          rel="noopener noreferrer"
        >
          {checkSumAddress}
        </a>
      </p>
    </div>
  );
}
